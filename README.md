# CLFW #

A Simple C and C++ based OpenCL Framework which provides custom C++ bindings to OpenCL C API Specification by Khronos

Currently runs on Linux and Windows

Tested On - 

OS - 
1) Windows 11 64-bit
2) Arch Linux
3) Kubuntu 22.04

GPU - 
1) AMD Radeon RX6600
2) NVIDIA GeForce RTX 3060
3) NVIDIA GeForce GTX 1650
